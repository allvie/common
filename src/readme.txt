This folder contains the source code for NanoByte.Common.

The "build.cmd" script will compile the code using the Visual Studio 2010/2012/2013 build environment on Windows.
The "build.sh" script will compile the code using Mono's xbuild on Unix-like systems.

Use "NanoByte.Common_VS2010.sln" to open the solution in Visual Studio 2010 (Service Pack 1 recommended).
Use "NanoByte.Common_VS2012.sln" to open the solution in Visual Studio 2012 oder 2013.
Use "NanoByte.Common_MonoDevelop.sln" to open the solution in MonoDevelop 2.4 or newer or Xamarin Studio or to compile with xbuild.
Important: Activate "Compile projects using MSBuild/XBuild" in the MonoDevelop/Xamarin Studio preferences!
